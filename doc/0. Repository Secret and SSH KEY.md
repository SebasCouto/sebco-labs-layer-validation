Vamos a retomar desde cero en Bash, paso a paso.

1. Generar las claves SSH en Bash (en Windows)
Primero, abre Git Bash en Windows (o cualquier terminal Bash que estés utilizando en Windows).

Usa este comando para generar un par de claves SSH:

bash
Copy
Edit
ssh-keygen -t rsa -b 4096 -C "tu_email@example.com"
Te pedirá la ubicación para guardar las claves, usa la ubicación predeterminada:

bash
Copy
Edit
/c/Users/TuUsuario/.ssh/id_rsa
Si ya tienes un par de claves, puedes sobreescribirlo o elegir otro nombre. Después, te pedirá una contraseña opcional.

Este comando generará dos archivos:

Clave privada: id_rsa (este es el archivo que nunca debes compartir).

Clave pública: id_rsa.pub (este es el archivo que puedes compartir, y se usa en GitHub).

2. ¿Dónde colocar la clave privada y la pública?
Clave privada (id_rsa): Esta debe ser usada en tu máquina local y guardada en el directorio .ssh de tu usuario. En el caso de GitHub Actions, la clave privada es la que se coloca en los secrets de tu repositorio.

Clave pública (id_rsa.pub): La clave pública es la que se debe agregar a GitHub como Deploy Key o en tu perfil en GitHub.

3. ¿Dónde colocar cada clave?
Clave privada (id_rsa) en GitHub Actions: Debes agregar la clave privada en los secrets de tu repositorio de GitHub. El secret puede llamarse, por ejemplo, SSH_PRIVATE_KEY, y luego la usas en tu workflow.

Clave pública (id_rsa.pub) en GitHub: La clave pública la debes agregar a tu cuenta de GitHub como una Deploy Key o a tu perfil de usuario en GitHub.

4. Obtener las claves
Si ya tienes las claves generadas, puedes verlas y copiarlas con los siguientes comandos:

Ver la clave privada:

bash
Copy
Edit
cat ~/.ssh/id_rsa
Ver la clave pública:

bash
Copy
Edit
cat ~/.ssh/id_rsa.pub
Si las claves están en otra ruta, simplemente cambia la ruta en los comandos anteriores.

5. Usar la clave privada en GitHub Actions
Tu archivo .yml de GitHub Actions debería verse más o menos así (en la parte donde configuras las claves):

yaml
Copy
Edit
      - name: Setup GitHub SSH key
        run: |
          mkdir -p "$HOME/.ssh"
          echo "$SSH_PRIVATE_KEY" > "$HOME/.ssh/id_rsa"
          chmod 600 "$HOME/.ssh/id_rsa"
          ssh-keyscan -H github.com >> "$HOME/.ssh/known_hosts"
En este caso, $SSH_PRIVATE_KEY es el secret en GitHub donde guardaste la clave privada.
